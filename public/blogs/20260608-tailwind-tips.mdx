用了 Tailwind CSS 之后，基本告别了手写 CSS 文件。但刚开始用的时候，很容易写出冗长的类名。下面分享几个实用技巧。

第一，善用 `group` 和 `group-hover`。比如卡片悬停时让子元素也跟着变化，不需要写任何 JS。第二，`peer` 选择器在处理表单状态时特别好用，比如输入框聚焦时改变相邻标签的颜色。

第三，别在 HTML 里堆太多类名，可以把重复的组合抽成组件。第四，`@apply` 指令适合封装可复用的样式模式，但不要滥用，否则就失去了 Tailwind 的原子化优势。

最后，把品牌色定义成 CSS 变量，在 Tailwind 配置中引用，这样换主题色只需要改一处。
