做个人项目最头疼的就是后端。搭数据库、写 API、管权限，一套下来几天就过去了。后来发现了 Supabase，这个问题彻底解决了。

Supabase 是一个开源的后端即服务平台，核心是 PostgreSQL 数据库。它自带 RESTful API 和实时订阅功能，创建数据表后自动生成接口，根本不用写后端代码。权限管理用 RLS（行级安全策略），在 SQL 层面控制谁能读什么数据，安全性很高。

更惊喜的是它内置了用户认证系统，支持邮箱密码登录、GitHub OAuth，几行代码就能集成到前端。免费额度对于个人项目绰绰有余，数据库有 500MB 空间、每月 5GB 流量。

如果你也在做个人项目，强烈建议试试 Supabase，真的能把开发效率提升一大截。
